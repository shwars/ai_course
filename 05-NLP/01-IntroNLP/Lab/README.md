# Работа с сервисами обработки текста: сокращение классики

## Задание

Сгенерируйте сокращенную версию нескольких глав одного из классических романов на русском или английском языках. Оцените результат и напишите осмысленный вывод.

Тексты романов можно найти на сайтах:

* [Project Gutenberg](https://www.gutenberg.org/) (на английском языке)
* [lib.ru](http://lib.ru) (на русском языке)

Для сокращения текста используйте сервисы:
* [MeaningCloud](https://www.meaningcloud.com/) (для английского и русского языков)
* [Sber Суммаризатор](https://sbercloud.ru/ru/datahub/rugpt3family/summarizer) (для русского языка)

Для реализации алгоритма используйте MS PowerAutomate или программу на Python.

Текст сокращайте по абзацам, короткие абзацы (диалоги) - пропускайте. Для каждого сокращенного абзаца оцените, насколько хорошо сделано сокращение, по шкале 1-5, и подведите итог работы всего сервиса.
